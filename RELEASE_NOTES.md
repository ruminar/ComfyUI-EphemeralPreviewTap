# 0.1.0

初回リリース。

## 追加

- `IMAGE` を受け取り、変更を加えずそのまま通過させる `Ephemeral Preview Tap` ノードを追加。
- 入力された画像を一時的なプレビューとして表示。
- ComfyUI の temp/output フォルダにはプレビュー画像を保存しない。
- 複数枚の画像が渡された場合、最大64枚までを1枚のコンタクトシートとして表示。
- 画像は縦横それぞれ50%に縮小し、長辺512pxを上限として表示。
- プレビューは現在実行中のクライアントにのみ送信。
- 各パラメータは固定。
  - JPEG quality: `80`
  - JPEG optimize: `false`
  - Gap: `6`
  - Max images: `64`
